如何挂到你的 GitHub 并访问
  1. 把本文件推送到你的仓库的 docs/stock-tool/index.html（与当前 AKShare 仓库路径一致）。
  2. 打开仓库 Settings → Pages，Source 选分支（如 main），文件夹选 /docs，保存。
  3. 几分钟后访问：
    https://<你的用户名>.github.io/<仓库名>/stock-tool/
    （若仓库名是 username.github.io 的特殊站，路径规则以 GitHub 文档为准。）

若东财、腾讯均在当前网络不可用，只能换网络/VPN 或在本地用 AKShare 拉数。
